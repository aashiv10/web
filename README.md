# Employee Management System (ASP.NET Web Forms)

This project is a simple Employee Management System built using:

- ASP.NET Web Forms
- C#
- SQL Server
- ADO.NET

## Features
- User Login Authentication
- Add Employee
- Edit Employee
- Delete Employee
- View Employee List

## Setup

1. Clone the repository
2. Open the solution in Visual Studio
3. Run `DB_Create.sql` in SQL Server
4. Update the connection string in `Web.config`
5. Run the project

## Tech Stack
- ASP.NET Web Forms
- C#
- SQL Server
- Visual Studio
